Perform a **ripemd160** of Base16 data.

### ripemd160

calculate ripemd160 hash of a base16 data.

```bash
~ qx ripemd160 --help

Usage: qx ripemd160 [hexstring]
```

#### Example

```bash
~ qx ripemd160 900df00d

31589998e7e92e769bfd5d453d12fbfa17c86297
```
